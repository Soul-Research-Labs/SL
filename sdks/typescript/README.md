# TypeScript SDK (Legacy)

> **Canonical location**: [`sdk/`](../../sdk/)
>
> The primary TypeScript SDK lives at the repository root under `sdk/`. This
> `sdks/typescript/` directory exists as a future namespace for additional
> language-specific SDKs (see `sdks/python/` for the Python SDK).
>
> Do **not** add new TypeScript SDK code here — use `sdk/` instead.
